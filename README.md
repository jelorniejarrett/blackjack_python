# Blackjack Game (Python)

A Blackjack game made using the command-line interface with Python. This Blackjack game has rules similar to those played in real life, featuring player and dealer gameplay.

## Features
- Card deck creation
- System for random card selection
- Decision-making by the player (hit/stand)
- Automated dealer behavior (AI)
- Winning and losing results
- Game replays

## Coding Language(s) Used
- Python

## Game Mechanics
- Two cards are dealt to both players
- The player chooses between hitting and standing
- The dealer plays according to predefined criteria
- Winner determination through comparison of scores

## Computer Science Principles Demonstrated
- Functionality and code modularity
- Looping and branching
- Random numbers generation
- Data list operations

## Potential Enhancements
- Incorporating a betting system
- Dynamic management of Ace value (1/11)
